# Workflow status

Dette prosjektet er gjennomført iterativt, med overlapp mellom analyse, planlegging og implementering.

Implementasjon ble brukt tidlig for å utforske tekniske muligheter og validere valg. Samtidig ble det tydelig at manglende tidlig og strukturert dokumentasjon førte til mer omfattende dialog med KI i implementeringsfasen. Uten et samlet og tydelig dokumentgrunnlag å støtte promptene på, krevde utviklingen flere iterasjoner og mer manuell oppfølging for å oppnå ønsket resultat.

Erfaringen har vist at tydelige artefakter som PRD og epics ikke bare er viktige for menneskelig forståelse, men også fungerer som effektive kontekstbærere i KI-assistert utvikling. Et mer ferdigstilt dokumentasjonsgrunnlag tidligere i prosessen ville sannsynligvis gjort implementeringen mer effektiv.
